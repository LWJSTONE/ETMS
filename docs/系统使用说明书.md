## 系统概述

### 系统简介

企业员工培训管理系统（ETMS）是一套功能完善的企业级培训管理平台，帮助企业实现：

- 员工培训全流程数字化管理
- 在线学习与考试考核一体化
- 培训数据可视化分析
- 多终端访问支持

### 系统角色

| 角色 | 权限说明 |
|------|---------|
| 超级管理员 | 拥有系统全部权限，可进行系统配置和用户管理 |
| 培训管理员 | 管理课程、培训计划、考核、签到等培训业务 |
| 部门主管 | 管理本部门员工培训，查看部门培训数据 |
| 普通员工 | 参与培训学习、签到打卡、在线考试 |

## 管理员端功能

### 用户管理

#### 功能说明

管理系统所有用户，包括员工信息的增删改查、导入导出、密码重置等。

#### 操作步骤

**新增用户：**

1. 点击左侧菜单【系统管理】→【用户管理】
2. 点击右上角【新增】按钮
3. 填写用户信息：
   - 用户名（必填，唯一）
   - 真实姓名（必填）
   - 性别
   - 手机号（唯一）
   - 邮箱（唯一）
   - 所属部门
   - 所属岗位
   - 状态
4. 点击【确定】保存

**编辑用户：**

1. 在用户列表中找到目标用户
2. 点击【编辑】按钮
3. 修改需要更新的信息
4. 点击【确定】保存

**删除用户：**

1. 在用户列表中找到目标用户
2. 点击【删除】按钮
3. 确认删除操作

**重置密码：**

1. 在用户列表中找到目标用户
2. 点击【重置密码】按钮
3. 密码将重置为默认密码：123456

**分配角色：**

1. 在用户列表中找到目标用户
2. 点击【分配角色】按钮
3. 勾选需要分配的角色
4. 点击【确定】保存

**导入用户：**

1. 点击【导入】按钮
2. 下载导入模板
3. 按模板格式填写用户数据
4. 上传填写好的Excel文件
5. 系统自动导入数据

**导出用户：**

1. 设置筛选条件（可选）
2. 点击【导出】按钮
3. 系统生成Excel文件并下载

### 角色管理

#### 功能说明

管理系统角色，控制用户的访问权限。

#### 操作步骤

**新增角色：**

1. 点击【系统管理】→【角色管理】
2. 点击【新增】按钮
3. 填写角色信息：
   - 角色编码（唯一标识）
   - 角色名称
   - 角色描述
   - 数据范围（全部/本部门/本人）
4. 点击【确定】保存

**分配权限：**

1. 在角色列表中找到目标角色
2. 点击【权限】按钮
3. 勾选菜单权限和按钮权限
4. 点击【确定】保存

### 部门管理

#### 功能说明

管理组织架构，支持无限级部门嵌套。

#### 操作步骤

**新增部门：**

1. 点击【系统管理】→【部门管理】
2. 点击【新增】按钮
3. 填写部门信息：
   - 上级部门
   - 部门名称
   - 部门编码
   - 部门负责人
   - 排序号
4. 点击【确定】保存

**编辑部门：**

1. 在部门树中找到目标部门
2. 点击【编辑】按钮
3. 修改需要更新的信息
4. 点击【确定】保存

**删除部门：**

1. 确保部门下无用户
2. 点击【删除】按钮
3. 确认删除操作

### 课程管理

#### 功能说明

管理培训课程，包括课程信息、资源、审核等。

#### 操作步骤

**新增课程：**

1. 点击【培训管理】→【课程管理】
2. 点击【新增】按钮
3. 填写课程基本信息：
   - 课程名称
   - 课程编码
   - 课程分类
   - 课程描述
   - 课程目标
   - 课程时长
   - 难度等级
   - 适用对象
   - 前置要求
4. 上传课程资源：
   - 封面图片
   - 视频资源
   - 文档资源
   - PPT资源
5. 点击【确定】保存

**提交审核：**

1. 课程创建完成后，点击【提交审核】
2. 课程状态变为"待审核"
3. 等待管理员审核

**审核课程：**

1. 切换到"待审核"状态筛选
2. 点击目标课程的【审核】按钮
3. 选择审核结果（通过/驳回）
4. 填写审核意见
5. 点击【确定】完成审核

**上架课程：**

1. 审核通过的课程可以上架
2. 点击【上架】按钮
3. 课程状态变为"已上架"
4. 员工可以看到并学习该课程

**下架课程：**

1. 点击【下架】按钮
2. 课程状态变为"已下架"
3. 员工将无法看到该课程

### 培训计划管理

#### 功能说明

创建和管理培训计划，分配培训任务给员工。

#### 操作步骤

**创建培训计划：**

1. 点击【培训管理】→【培训计划】
2. 点击【新增】按钮
3. 填写计划基本信息：
   - 计划名称
   - 计划描述
   - 培训目标
   - 关联课程
   - 计划类型（必修/选修）
   - 开始日期
   - 结束日期
4. 设置培训对象：
   - 目标类型（部门/岗位/个人）
   - 选择具体对象
5. 设置考核要求：
   - 最低学习时长
   - 最低完成进度
   - 是否需要考试
   - 及格分数
   - 最大补考次数
6. 点击【确定】保存

**发布培训计划：**

1. 培训计划创建完成后，点击【发布】
2. 系统自动创建学习进度记录
3. 向目标用户发送培训通知
4. 计划状态变为"已发布"

**查看进度：**

1. 点击【查看进度】按钮
2. 查看各部门/人员的学习进度
3. 导出进度数据

### 题库管理

#### 功能说明

管理考试题目，支持多种题型。

#### 题型说明

| 题型 | 说明 | 答案格式 |
|------|------|---------|
| 单选题 | 四选一 | A/B/C/D |
| 多选题 | 多选 | ABC/ABD等 |
| 判断题 | 正误判断 | 正确/错误 |
| 填空题 | 填空 | 具体答案 |
| 简答题 | 简述 | 文字描述 |

#### 操作步骤

**新增题目：**

1. 点击【考核管理】→【题库管理】
2. 点击【新增】按钮
3. 选择题目类型
4. 填写题目内容
5. 填写选项（选择题）
6. 填写正确答案
7. 填写答案解析
8. 设置题目难度和分值
9. 关联课程（可选）
10. 点击【确定】保存

**批量导入：**

1. 点击【导入】按钮
2. 下载导入模板
3. 按模板格式填写题目数据
4. 上传Excel文件
5. 系统自动导入

### 试卷管理

#### 功能说明

组卷管理，支持手动组卷和随机组卷。

#### 操作步骤

**手动组卷：**

1. 点击【考核管理】→【试卷管理】
2. 点击【新增】按钮
3. 选择组卷方式：手动组卷
4. 填写试卷基本信息
5. 从题库中选择题目
6. 设置每题分值
7. 点击【确定】保存

**随机组卷：**

1. 点击【新增】按钮
2. 选择组卷方式：随机组卷
3. 填写试卷基本信息
4. 设置抽题规则：
   - 单选题数量
   - 多选题数量
   - 判断题数量
   - 各难度题目数量
5. 系统自动从题库随机抽题
6. 点击【确定】保存

**发布试卷：**

1. 试卷创建完成后，点击【发布】
2. 试卷可关联到培训计划
3. 学员可参加考试

## 员工端功能

### 我的课程

#### 功能说明

查看分配给自己的培训课程，参与在线学习。

#### 操作步骤

**查看课程列表：**

1. 点击【我的培训】→【我的课程】
2. 查看必修课程和选修课程
3. 查看学习进度

**开始学习：**

1. 找到目标课程
2. 点击【开始学习】按钮
3. 进入课程学习页面
4. 观看视频、阅读文档
5. 系统自动记录学习进度

**完成学习：**

1. 学习完成后，进度达到100%
2. 如果有考试要求，进入考试环节
3. 完成考试后，培训计划完成

### 我的考试

#### 功能说明

参加在线考试，查看考试记录。

#### 操作步骤

**参加考试：**

1. 点击【我的培训】→【我的考试】
2. 查看待参加的考试
3. 点击【开始考试】按钮
4. 阅读考试须知
5. 开始答题

**答题操作：**

1. 单选题：点击选项选择答案
2. 多选题：勾选多个选项
3. 判断题：选择正确/错误
4. 填空题：填写答案
5. 简答题：输入文字描述

**提交试卷：**

1. 确认所有题目已作答
2. 点击【提交】按钮
3. 确认提交操作
4. 系统自动计算客观题得分

**注意事项：**

- 注意考试时长
- 不要频繁切屏（超过次数自动提交）
- 答案会自动保存
- 超时自动提交

### 学习记录

### 我的成绩

查看考试成绩和历史记录。

#### 查看成绩

2. 查看所有考试成绩

## 常见操作指南

### 如何创建一个完整的培训流程

**步骤1：创建课程**

1. 进入【课程管理】→ 新增课程
2. 填写课程信息，上传资源
3. 提交审核并等待通过
4. 上架课程

**步骤2：创建题库**

1. 进入【题库管理】→ 新增题目
2. 创建各类型题目
3. 设置正确答案和解析

**步骤3：创建试卷**

1. 进入【试卷管理】→ 新增试卷
2. 手动或随机组卷
3. 发布试卷

**步骤4：创建培训计划**

1. 进入【培训计划】→ 新增计划
2. 关联课程和试卷
3. 设置培训对象
4. 发布培训计划

**步骤5：跟踪进度**

1. 进入【学习进度】查看学习情况
2. 进入【签到记录】查看出勤情况
3. 进入【成绩管理】查看考试成绩

### 如何导入大量用户

1. 进入【用户管理】
2. 点击【导入】→【下载模板】
3. 按模板格式填写用户数据
4. 保存Excel文件
5. 点击【导入】→ 选择文件
6. 系统自动校验并导入
7. 查看导入结果

### 如何批量分配培训任务

1. 进入【培训计划】→ 新增计划
2. 选择目标类型为"部门"
3. 勾选多个部门
4. 保存并发布
5. 系统自动分配给所有选中部门的员工

## 常见问题解答

### 登录相关

**Q: 忘记密码怎么办？**

A: 请联系系统管理员重置密码，重置后密码为：123456

**Q: 账号被锁定怎么办？**

A: 连续输错密码5次会锁定账号，请联系管理员解锁或等待30分钟后自动解锁

**Q: 登录后页面空白？**

A: 请清除浏览器缓存或尝试更换浏览器

### 学习相关

**Q: 学习进度不更新？**

A: 请确保完整观看视频，进度会在视频播放完成后更新

**Q: 课程无法播放？**

A: 请检查网络连接，或尝试更换浏览器

**Q: 学习时长统计不准确？**

A: 学习时长在页面关闭时统计，请正常退出学习页面

### 考试相关

**Q: 考试中途断网怎么办？**

A: 系统会自动保存答题进度，重新登录后可继续答题

**Q: 考试超时怎么办？**

A: 超时后系统自动提交试卷，请合理安排时间

**Q: 考试不及格怎么办？**

A: 根据培训计划设置，可能有补考机会，请查看培训计划详情

### 其他问题

**Q: 如何修改个人信息？**

A: 点击右上角头像 → 个人中心 → 修改信息

**Q: 如何查看通知消息？**

A: 点击右上角消息图标查看系统通知

**Q: 如何导出数据？**

A: 各列表页面均有导出按钮，点击即可导出Excel文件
