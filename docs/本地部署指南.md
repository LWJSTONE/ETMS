## 安装JDK

### Windows系统

#### 方法一：使用安装包（推荐新手）

1. **下载JDK**
   - 访问 Oracle 官网：https://www.oracle.com/java/technologies/downloads/
   - 选择 Java 17，点击 Windows 标签
   - 下载 `.exe` 安装包（如 `jdk-17_windows-x64_bin.exe`）

2. **安装JDK**
   - 双击下载的安装包
   - 点击"下一步"直到安装完成
   - 默认安装路径：`C:\Program Files\Java\jdk-17`

3. **配置环境变量**
   - 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
   - 点击"新建"系统变量：
     - 变量名：`JAVA_HOME`
     - 变量值：`C:\Program Files\Java\jdk-17`
   - 编辑系统变量 `Path`，添加：
     - `%JAVA_HOME%\bin`
   - 点击"确定"保存

4. **验证安装**
   - 打开命令提示符（按 Win+R，输入 `cmd`）
   - 输入命令：`java -version`
   - 显示版本信息表示安装成功

#### 方法二：使用Chocolatey（快速安装）

```powershell
# 以管理员身份运行PowerShell
choco install openjdk17 -y
```

### macOS系统

```bash
# 使用Homebrew安装
brew install openjdk@17

# 配置环境变量
echo 'export PATH="/usr/local/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# 验证
java -version
```

### Linux系统

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk -y

# CentOS/RHEL
sudo yum install java-17-openjdk-devel -y

# 验证
java -version
```

## 安装MySQL

### Windows系统

1. **下载MySQL**
   - 访问：https://dev.mysql.com/downloads/mysql/
   - 选择 Windows 版本，下载 MSI 安装包

2. **安装MySQL**
   - 双击安装包，选择"Developer Default"或"Server only"
   - 设置root密码（建议设置为 `root`，方便记忆）
   - 端口保持默认：`3306`
   - 完成安装

3. **验证安装**
   ```cmd
   mysql -u root -p
   # 输入密码后进入MySQL命令行
   ```

### macOS系统

```bash
# 使用Homebrew安装
brew install mysql

# 启动服务
brew services start mysql

# 设置密码
mysql_secure_installation
```

### Linux系统

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install mysql-server -y
sudo systemctl start mysql
sudo mysql_secure_installation

# CentOS/RHEL
sudo yum install mysql-server -y
sudo systemctl start mysqld
sudo mysql_secure_installation
```

## 安装Redis

### Windows系统

1. **下载Redis**
   - 访问：https://github.com/tporadowski/redis/releases
   - 下载最新的 `.zip` 文件

2. **安装Redis**
   - 解压到任意目录，如 `D:\redis`
   - 打开命令提示符，进入Redis目录

3. **启动Redis**
   ```cmd
   cd D:\redis
   redis-server.exe
   ```
   - 保持窗口打开，Redis服务正在运行

4. **验证安装**
   - 新开命令提示符窗口
   ```cmd
   cd D:\redis
   redis-cli.exe
   127.0.0.1:6379> ping
   PONG
   ```

### macOS系统

```bash
# 安装
brew install redis

# 启动服务
brew services start redis

# 验证
redis-cli ping
```

### Linux系统

```bash
# Ubuntu/Debian
sudo apt install redis-server -y
sudo systemctl start redis

# CentOS/RHEL
sudo yum install redis -y
sudo systemctl start redis

# 验证
redis-cli ping
```

## 配置数据库

### 创建数据库

1. **打开MySQL命令行或图形工具**

2. **创建数据库**
   ```sql
   CREATE DATABASE etms DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
   ```

3. **执行数据库脚本**
   
   方法一：使用命令行
   ```bash
   # 进入项目目录
   cd ETMS/etms-backend/src/main/resources
   
   # 执行建表脚本
   mysql -u root -p etms < schema.sql
   
   # 执行初始数据脚本
   mysql -u root -p etms < data.sql
   ```
   
   方法二：使用Navicat等图形工具
   - 打开Navicat，连接到MySQL
   - 右键数据库 `etms` → 运行SQL文件
   - 选择 `schema.sql` 执行
   - 选择 `data.sql` 执行

4. **验证数据导入**
   ```sql
   USE etms;
   SHOW TABLES;
   SELECT * FROM sys_user;
   ```

### 修改后端配置

1. **打开配置文件**
   - 文件路径：`etms-backend/src/main/resources/application.yml`

2. **修改数据库配置**
   ```yaml
   spring:
     datasource:
       url: jdbc:mysql://localhost:3306/etms?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
       username: root          # 你的MySQL用户名
       password: root          # 你的MySQL密码
   ```

3. **修改Redis配置**
   ```yaml
   spring:
     redis:
       host: localhost         # Redis地址
       port: 6379              # Redis端口
       password:               # Redis密码（如果没有设置密码，留空）
   ```

## 启动后端

### 方法一：使用IDEA启动（推荐）

1. **用IDEA打开后端项目**
   - 打开IDEA
   - 选择"Open"，找到 `ETMS/etms-backend` 目录
   - 选择"Maven项目"，点击确定

2. **等待依赖下载**
   - IDEA会自动下载Maven依赖
   - 右下角显示下载进度
   - 等待下载完成（首次可能需要几分钟）

3. **启动项目**
   - 找到主类：`src/main/java/com/etms/EtmsApplication.java`
   - 右键 → "Run 'EtmsApplication'"
   - 控制台显示启动日志

4. **启动成功标志**
   ```
   ======================================
     企业员工培训管理系统启动成功!
     访问地址: http://localhost:8080/api
   ======================================
   ```

### 方法二：使用命令行启动

```bash
# 进入后端目录
cd ETMS/etms-backend

# 编译项目（首次运行或代码修改后）
mvn clean package -DskipTests

# 启动项目
java -jar target/etms-backend-1.0.0.jar

# 或者直接运行
mvn spring-boot:run
```

### 验证后端启动

1. **访问Swagger文档**
   - 打开浏览器访问：http://localhost:8080/api/swagger-ui.html
   - 能看到API文档页面表示启动成功

2. **测试登录接口**
   - 使用Postman或Apifox
   - POST请求：http://localhost:8080/api/auth/login
   - Body（JSON）：
     ```json
     {
       "username": "admin",
       "password": "123456"
     }
     ```

## 常见问题

### 1. 后端启动失败

#### 问题：端口8080被占用

**Windows解决方法：**
```cmd
# 查看占用8080端口的进程
netstat -ano | findstr :8080

# 结束进程（PID是上一步查到的进程ID）
taskkill /PID 进程ID /F
```

**Linux/Mac解决方法：**
```bash
# 查看占用8080端口的进程
lsof -i :8080

# 结束进程
kill -9 进程ID
```

#### 问题：数据库连接失败

1. 检查MySQL服务是否启动
2. 检查数据库用户名密码是否正确
3. 检查数据库 `etms` 是否已创建
4. 检查防火墙是否开放3306端口

#### 问题：Redis连接失败

1. 检查Redis服务是否启动
2. 检查Redis配置是否正确

### 2. 前端启动失败

#### 问题：npm install失败

```bash
# 清除npm缓存
npm cache clean --force

# 删除node_modules目录
rm -rf node_modules

# 使用国内镜像重新安装
npm install --registry=https://registry.npmmirror.com
```

#### 问题：启动后页面空白

1. 检查后端是否正常启动
2. 检查浏览器控制台是否有错误
3. 检查网络请求是否正常

### 3. 登录失败

#### 问题：登录提示密码错误

1. 确认数据库初始数据已正确导入
2. 确认用户名和密码正确
3. 默认密码是 `123456`

#### 问题：登录后跳转空白

1. 检查浏览器控制台错误信息
2. 检查后端API是否正常返回数据
3. 清除浏览器缓存重试

### 4. 其他问题

#### 问题：Maven依赖下载慢

修改Maven配置文件 `settings.xml`，添加国内镜像：
```xml
<mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

#### 问题：前端接口请求跨域

后端已配置CORS，如果仍有问题，检查：
1. 前端请求地址是否正确（`/api`前缀）
2. 后端是否正常启动
